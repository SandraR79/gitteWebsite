29.juni 1946 // Gitte erblickt in Aarhus (Dänemark) als zweite Tochter von Erna und Otto Haenning-Johannsson das Licht der Welt.

1954 // Vater Otto überredet die kleine Gitte mit ihm „Giftes med farmand“ einzusingen. Dies wird ein großer Erfolg und Gitte über Nacht zum Kinderstar.

1958 // Gitte erhält ihren ersten eigenen Plattenvertrag und macht Karriere ohne Vater Otto.

1960 // Nicht nur durch die Arbeit mit Jazzgrössen wie / Gittes große Liebe - niemals wurde sie verspielt, auch nicht in Monte Carlo!

1960-62 // Musicals, Shows und Filme / Vielseitig und breigefächert aufgestellt - bis heute!

1962 // Gitte nimmt an der dänischen Vorentscheidung zum Grand Prix d'Eurovision de la Chanson (ESC) mit „Jeg snakker med mig selv“ teil. Das Lied wird disqualifiziert.

1963 // „Ich will'nen Cowboy als Mann“ gewinnt bei den Deutschen Schlagerfestspielen in Baden-Baden und Gittes Karriere in Deutschland beginnt. Cowboy - Fluch oder Segen / Ihre absolute Visitenkarte, Türöffner und sicherlich auch Teufelsbeitrag

1964-65 // 1964-65 Rex und Gitte / Traumpaar der 60s - und doch von allem zuviel

1966 // neue Wege - trotzdem kein gefallen an Liedern für sie. Problem mit Gietz / Neuer Start - alter Produzent, progressiver neuer Musikstilversuch

1968 // 1968 Der rote Mantel / So wie Gott sie schuf auf der großen Leinwand - in Dänemark nothing new - in Deutschland ein Skandal

1969 // Jazzplatte als Lockmittel / Wieder die große Liebe - große Produktion mit dem who-is-who der Branche - kein Erfolg beim "alten" Publikum

1971-72 // Rückzug aus Musikgeschäft/ umzug rom / Pause vom großen Musikbusiness und Besinnung auf das Leben

1973 // 1973 Grand prix Teilnahme / Nochmal ESC-Feeling, diesmal für Deutschland. 8.Platz am Ende, jedoch nicht so erfolgreich wie Katja Ebstein und Mary Roos zuvor.

1974 // Eigene TV-Show „Begegnung mit Gitte"

1974-75 // Neuer Erfolge mit Hits/ showauftritte / Trotz "ESC-Pleite" wieder in aller Munde und präsenter als jemals zuvor

1974 // Heirat aus Trotz / Man muss schließlich auch mal "ja" sagen, Heirat aus Trotz mit Joe Geistler weil alle was dagegen zu sagen hatten

1976 // Scheidung / Und dann doch "Schluss mit lustig"

1978 // Wonderful Gitte aus Kopenhagen / Ritterschlag mit großer Personalityshow im ZDF zur Hauptsendezeit

1980-81 // „Imagewechsel“ und erneuter Plattenfirma nach Wechsel zu Global anspruchsvollere Musik / Nach erneuter Pause und Kraftschöpfen ein Neustart, ganz neu, ganz anders

1981 // Deutscher Schallplattenpreis / Die Früchte des Imagewechsels - der größte deutsche Musikpreis für das Konzeptalbum "Bleib noch bis zum Sonntag"

1983 // Das Album "Berührungen" wird einer ihrer größten Erfolge. Fortan unter dem Namen Gitte Haenning

1984 // Gitte bekommt ihre bisher einzige Goldene Schallplatte für das Erfolgsalbum „Berührungen“ und geht damit auf ihre erste Tournee.

1985 // Ein Hundebiss ins Gesicht zwingt Gitte zur einjährigen Zwangspause.

1987-88 // Jetzt erst recht Album. Tournee / Genesen und gestärkt - wieder einmal Neustart.

1991-92 // Friedrich Kurz, der Musicalpapst tritt in ihr Leben und es wird Liebe

1993 // Neue CD Liebster (supertoll, aber nicht gut vermarktet) / Plattenfirmawechsel, neues Produktionsteam. Leider wurde das wunderbare Album nicht ausreichend promotet.

1994 // „Mitten in der Nacht“ Tour

1995 // Shakepeare und Rock 'n' Roll / Friedrich Kurz' Produktion mit Gitte in der 1.Besetzung

1997 // Gitte bringt ihre große Liebe zum Jazz in ihre Konzerte „Songs for my father“ in der Komischen Oper Berlin wird ein großer Erfolg

1998 // Gitte singt ihre internationalen Lieblingslieder ein, inkl. Titelsong für die ZDF-Serie "Lisa Falk", CD wurde leider kein großer Publikumserfolg

2001-02 // In Jazz tour / Umjubelte Konzertreihe, immer und immer wieder verlängert und Wiederholungstermine in gleichen Venues

2004 // Neue CD "Johansson" / Erstes selbstproduziertes Album zum 50. Bühnenjubiläum

2004 // CD "In Jazz" - zusätzlich zum neuen Studioalbum wird die Live-Aufnahme zur umjubelten Tournee veröffentlicht.

2004-07 // Gitte, Wencke, Siw - Die Show / Ursprünglich als 3-wöchige Sommershow in Berlin geplant - am Ende Konzerte in der gesamten Republik mit über 500 Shows. Herausragende Kartenabsätze.

2006 // Doku Gitte / Die Gitte Hænning-Story - 2stündiger Biografie-Dokufilm.

2007-2010 // Tourneen, mal mehr Jazz, mal mehr Pop-orientiert

2010 // Album "Was ihr wollt" - 8 neuproduzierte Hits und 4 neue Songs werden veröffentlicht

2010 // Theatertour mit dem Shakespeare-Stück "Was ihr wollt"

2012 // Teilnahme an der RTL-Tanzshow "Let's dance"

2014-2015 // Theatertour mit dem Stück "Loveletters"

2016 // Sunset Boulevard / Hauptrolle als Norma Desmond in "Sunset Boulevard" im Theater Lübeck

2018 // Flashdance / Erneute Rolle im Musical. Im Hamburger Mehr!-Theater am Großmarkt war sie als alternde Tanzlehrerin Hannah in der Bühnenversion des Kultfilms der achtziger Jahre Flashdance in einer Sprechrolle zu sehen
